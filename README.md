# 4×4 DRAM Memory Design and Simulation

This project implements a transistor-level 4×4 Dynamic Random Access Memory (DRAM) array using LTSpice. The design models key DRAM operations including row activation, sense amplification, row buffering, column selection, destructive-read recovery, and periodic refresh cycles.

The objective was to create a functional memory subsystem capable of performing realistic DRAM read operations while analyzing timing performance and memory refresh behavior.


## Features

- 4×4 transistor-level DRAM array
- Row and column address decoding
- Sense amplifier implementation
- Row buffer using D flip-flops
- Destructive read recovery through writeback
- Automatic refresh after every 5 memory operations
- FSM-based control sequencing
- Read latency measurement and timing optimization
- LTSpice simulation and waveform verification
